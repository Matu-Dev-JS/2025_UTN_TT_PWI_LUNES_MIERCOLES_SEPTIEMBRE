npm viene instalado con node.js y es el node package manager. Nos permitira manejar los paquetes de node.

Para instalar las dependencias debemos ejecutar el comando npm install (en la carpeta raiz del proyecto).
Las dependencias se instalaran en la carpeta node_modules.

Nuestro proyecto de react no funciona con live-server, para poder tener un servidor que permita ejecutar nuestra app en nuestra pc usaremos el paquete Vite. (Vite Server)
